# Link Shortener

A simple, fast URL shortening service with analytics.

## Quick Start

```bash
# Open in AI-assisted coding mode
./start-opencode.sh   # Opens in plan mode
./start-claude.sh     # Opens Claude Code

# Or manually
npm install
npm run dev
```

## Stack

- Next.js 14 (App Router)
- Neon Postgres
- Railway deployment

## Features

- Shorten URLs with custom or auto-generated slugs
- Click analytics (referrer, geo, time)
- Fast redirects (<100ms)
