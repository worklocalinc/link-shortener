# Link Shortener

A modern URL shortener with analytics and QR codes.

## Quick Start

```bash
npm install
npm run db:push
npm run dev
```

Open http://localhost:3000

## Features

- Shorten long URLs to memorable links
- Custom slug support
- QR code generation
- Click analytics
- Fast redirects

## Tech Stack

- Next.js 14 + React
- Neon Postgres
- Tailwind CSS
- Railway / Dev Server
